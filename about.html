Coś o mnie

Nazywam się Michał. Jestem początkującym frontend developerem z pasją do tworzenia intuicyjnych i nowoczesnych aplikacji webowych. Skupiam się na budowaniu przejrzystych interfejsów i dbam o czysty, dobrze zorganizowany kod. Potrafię Tworzyć responsywne strony za pomocą HTML, CSS oraz JavaSript. Potrafie także korzystać z systemu kontroli wersji(Git) oraz potrafię obsługiwać podstawowe funkcje w systemie linux. W niedalekiej przyszłości planuję poszerzyć moją wiedzę o biblioteki takie jak React oraz tailwind CSS
